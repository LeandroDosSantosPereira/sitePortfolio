sobre

Olá me chamo Leandro e atualmente estou cursando o quinto semestre do curso de Análise e desenvolvimento de sistemas, pelo Instituto federal de Mato Grosso do Sul, Câmpus de Nova Andradina. Sou um entusiasta das tecnologias web e apaixonado pelo desenvolvimento de games. Neste site está disponível alguns dos meus projetos.

Habilidades

HTML+CSS+JAVASCRIPT

São as linguagens base do desenvolvimento web, que em conjunto são reponsáveis pela estrutura, estilo e interatividade as aplicações com elas desenvolvidas.

Gdevelop

Engine para desenvolvimento de games 2D, multiplataforma de código aberto, permitindo exportar seus jogos para IOS, Android, Desktop e Web.

Inkascap

Inkscape é um editor de gráficos vetoriais de código-fonte aberto semelhante ao Adobe Illustrator, Corel Draw, Freehand ou Xara X. O que diferencia o Inkscape é o uso nativo de Scalable Vector Graphics (SVG), em português Gráficos Vetoriais Escaláveis, um padrão aberto baseado em XML do consórcio W3C.
